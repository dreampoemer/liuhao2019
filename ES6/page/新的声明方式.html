ES6声明变量的六种方法，其中ES5只有1，2两种声明变量的方法。
  1. var命令 
                        
                        var a=2;            
                        {
                        var a=3;
                        }
                        console.log(a);//3----var是全局声明的

                        var a=2;
                        {
                        let a=3;
                        }
                        console.log(a);//2----let声明只在区块内起作用，外部是不可以调用的

                        const a="JSPang";
                        var a='技术胖';
                        console.log(a);//报错：'Identifier 'a' has already been declared'。因为const声明的变量是不可改变的。
